__Тема проекта:__  

_"Генератор паролей. Система позволяет генерировать пароли различной длины, используя различные алгоритмы. 
Система хранит пароли в зашифрованном виде, а также хранит историю их формирования."_  

__Текущие задачи:__  
- произвольный пользователь может зарегистрироваться в системе
(реализован шаблон регистрации и контроллер);
- пароли новых пользователей хранятся в зашифрованном виде;
- неавторизованному пользователю не доступны ресурсы приложения
  и в случае попытки получить доступ осуществляется переход на
  страницу авторизации;
- swagger-ui доступен только пользователю с ролью ADMIN;
- пользователь может выйти из системы;
