__Тема проекта:__  

_"Генератор паролей. Система позволяет генерировать пароли различной длины, используя различные алгоритмы. 
Система хранит пароли в зашифрованном виде, а также хранит историю их формирования."_  

__Текущие задачи:__  
- методы репозиториев доступны по REST API;
- реализованы REST и HTML контроллеры для кастомных методов;
- реализована унифицированная обработка ошибок;
- подключение Swagger UI;
