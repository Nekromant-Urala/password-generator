__Тема проекта:__  

_"Генератор паролей. Система позволяет генерировать пароли различной длины, используя различные алгоритмы. 
Система хранит пароли в зашифрованном виде, а также хранит историю их формирования."_  

__Текущие задачи:__  
- спроектировать ER-диаграмму;
- реализовать структуру БД в коде;
- для каждой из сущностей реализованы репозитории;
- реализованы 2 метода в репозиториях с использованием именованных запросов HQL, и реализованы аналогичные запросы с использованием Criteria API;
- реализован сервис с транзакционной операцией;
- написаны автотесты на кастомные методы репозитория и на транзакционный метод сервиса;
