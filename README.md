__Тема проекта:__  

_"Генератор паролей. Система позволяет генерировать пароли различной длины, используя различные алгоритмы. 
Система хранит пароли в зашифрованном виде, а также хранит историю их формирования."_  

__Текущие задачи:__  
- отчет формируется в формате HTML и удовлетворяет всем
  требованиям из раздела “Ход работы” пункта 1;
- реализована сущность “Отчет” и репозиторий к ней;
- реализован сервис для работы с отчетом;
- сервис содержит методы для создания и получения отчета, а также
  асинхронный метод формирования отчета в многопоточном режиме;
- реализован рест контроллер для работы с отчетами.
