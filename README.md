__Тема проекта:__  

_"Генератор паролей. Система позволяет генерировать пароли различной длины, используя различные алгоритмы. 
Система хранит пароли в зашифрованном виде, а также хранит историю их формирования."_  

__Текущие задачи:__  
- реализация класса сущности БД;
- реализация бина для имитации работы БД;
- реализация слоя доступа к данным БД;
- реализация бизнес-логики приложения;
- реализация консольного ввода;
- конфигурация приложения;
- мониторинг приложения.
